OPCIÓN A: Deploy en Vercel (MÁS FÁCIL - 2 minutos)
───────────────────────────────────────────────────
  npm install -g vercel
  vercel --prod
  ✅ Listo! Tu sitio estará en: https://tu-proyecto.vercel.app


OPCIÓN B: Subir a GitHub y Auto-Deploy (3 minutos)
───────────────────────────────────────────────────
  git add .
  git commit -m "KSC Website - Production ready"
  git push origin master
  → Luego conectar con Vercel desde dashboard


OPCIÓN C: Docker (5 minutos)
───────────────────────────────────────────────────
  docker-compose up -d
  ✅ Listo! Tu sitio estará en: http://localhost:3000
