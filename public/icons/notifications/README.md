# Notification Icons

Esta carpeta contiene los iconos SVG para el componente de notificaciones.

## Archivos necesarios:

1. **success.svg** - Icono de éxito/confirmación (verde)
2. **error.svg** - Icono de error (rojo)
3. **info.svg** - Icono de información (azul)
4. **warning.svg** - Icono de advertencia (amarillo)
5. **close.svg** - Icono para cerrar notificaciones (gris)

## Uso:

Los iconos se utilizan en el componente `Notification.tsx` ubicado en:
`src/components/ui/Notification.tsx`

## Formato:

- Formato: SVG
- Tamaño recomendado: 24x24px
- Color: Debe ser fácilmente modificable con CSS/Tailwind
- Optimizado: Sin metadatos innecesarios
