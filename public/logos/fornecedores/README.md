## Especificaciones Recomendadas:

- **Formato**: PNG con fondo transparente
- **Dimensiones**: 200x80px (ancho x alto) o proporcional
- **Tamaño**: Menos de 100KB por imagen
- **Calidad**: Alta resolución para pantallas retina

## Nota:

Reemplaza cada archivo placeholder con el logo oficial del proveedor.
Los logos se mostrarán al lado del título de cada tarjeta de proveedor.
